# Homebrew Tap for claude-code-starter

Production-ready Claude Code configuration.

## Installation

```bash
brew tap zbruhnke/claude-code-starter
brew install claude-code-starter
```

## Usage

```bash
claude-code-starter help
claude-code-starter init
claude-code-starter adopt skills
```

## Update

```bash
brew upgrade claude-code-starter
```
